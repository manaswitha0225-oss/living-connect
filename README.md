# living-connect
The Apartment Management System is a web-based application designed to simplify and automate the daily operations of residential apartment complexes. It provides an efficient platform for managing residents, apartments, maintenance requests, payments, and administrative tasks in a centralized system.

This system allows administrators to manage apartment details, resident records, rent payments, and service requests with ease. Residents can submit maintenance complaints, track payment history, and receive important notifications, improving communication and transparency between management and tenants.

The application aims to reduce manual work, minimize errors, and improve overall efficiency in apartment administration by providing a secure, user-friendly, and scalable solution.

Key Features

Resident and apartment management

Rent and payment tracking

Maintenance and complaint management

Admin and user role management

Secure authentication and authorization

Dashboard with real-time data overview
